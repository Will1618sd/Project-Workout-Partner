# Project 5D Workout partner

## Description
Workout Partner is an Android app for sport support. It keep tracks of your indoor and outdoor activities. Distance, velocity and even a map of your activity is available for Running, Cycling and Swimming; and the number of repetitions and sets for Workout session, with different types of exercises (bench press, squat, and deadlift). 

Using a SmartWatch? --> An automatic count of the repetitions allows you to perform your workout without even having to think of it. 

Want some challenge? --> Weekly challenges will be proposed to you. And you can also add your friends to see their recent activities (and motivate yourself to get on track).

## Installation
This project is mostly using Kotlin. Android Studio may be usefull to modify, emulate and debug the project. The app can be installed by generating the APK files on Android Studio.
